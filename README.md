CAJERO JSF
Este es un proyecto jsf que imita las funciones basicas de un cajero en linea.

--Instrucciones--
Se debe Ejecutar En Eclipse IDE, por compatibilidad.
1. Clonar repositorio
2. Importar en Eclipse como proyecto Maven existente
3. Configurar Tomcat 11
4. Ejecutar el index.xhtml

--Funcionalidades--
- Inicio de sesión con PIN
- Depositos (con validacion de entrada)
- Retiros (con validacion de saldo)
- Consulta de saldo
